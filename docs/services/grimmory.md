# Grimmory

<img src="/docs/images/services/grimmory.svg" alt="Grimmory Logo" />

## What is Grimmory?

Grimmory is a self-hosted ebook library manager and reader, and the successor 
to Booklore. It allows you to organize, manage, and read your ebook collection 
from a clean web interface, with support for auto-importing books via a 
bookdrop folder.
